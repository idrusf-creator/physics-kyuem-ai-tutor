# Physics @ KYUEM AI Tutor v2

This is a ready-to-embed AI chatbot package for Google Sites.

## Files included

- `index.html`  
  The chatbot page.

- `google_apps_script_proxy.gs`  
  The secure backend proxy for OpenAI.

- `GOOGLE_SITES_EMBED_GUIDE.md`  
  Step-by-step guide for deployment and embedding.

## Main features

- Student Mode
- Teacher Mode
- CAIE 9702 topic selector
- Paper 1, 2, 3, 4 and 5 support
- Quick prompts
- Worked solution format
- Teacher resource generation
- Copy last reply
- Clear chat

## Important

Do not place your OpenAI API key inside `index.html`.

Use the Google Apps Script backend.
